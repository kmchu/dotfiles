# dotfiles
Personal configuration files for MacOS.

## Installaton
1. Setup ssh access to this repo
1. Copy `.bin/install.sh` onto local system
1. Run script
1. Install `bash-completion` with `brew`

## Update VIM plugins
1. Open NeoVim
1. Run `:PlugInstall`
1. Optionally, you can update plugins using `:PlugUpdate`
